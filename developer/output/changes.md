# Исправление бага классификации "Підбери товар"

## Проблема
Запросы типа "Підбери товар" + текст про роутер уходили в FAQ вместо product-agent из-за отсутствия императивных форм глаголов в паттерне product_selection_pattern.

## Корневая причина
В файле `orchestrator/classification/pattern_detector.py` в product_selection_pattern (priority 77) последний regex паттерн содержал только инфинитивные формы глаголов:
- `підібрати`, `подобрать`, `вибрати`, `выбрать`, `порекомендуй`, `порадь`, `посоветуй`

Но НЕ содержал императивные формы:
- `підбери`, `подбери`, `выбери`, `вибери`

## Решение
Добавлены недостающие императивные формы глаголов в последний regex паттерн product_selection_pattern:

```python
# Было:
r'\b(підібрати|подобрать|вибрати|выбрать|порекомендуй|порадь|посоветуй)\b'

# Стало:
r'\b(підібрати|подобрать|вибрати|выбрать|порекомендуй|порадь|посоветуй|підбери|подбери|выбери|вибери)\b'
```

## Измененные файлы
- `/home/aimchn/Desktop/Comfy/Multi Agent System/comfy-agents/comfy-agents/orchestrator/classification/pattern_detector.py`

## Результат
Теперь запросы "Підбери товар", "Подбери роутер", "Выбери телефон", "Вибери ноутбук" будут корректно направляться к product-agent вместо FAQ.
