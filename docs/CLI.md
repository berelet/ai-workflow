# CLI — Запуск AI-агентов для проекта

## Назначение

Запускает kiro-cli с полным контекстом проекта: backlog, pipeline, инструкции агента, skills.

## Команда

```bash
python cli.py
```

Прямой запуск:
```bash
python cli.py <проект> --task "задача" --stage PM
```

## Работа в терминале

### Меню

```
╭───────────────────────────────────╮
│     AI Workflow CLI               │
╰───────────────────────────────────╯

  Выбери проект:

   1) ai-workflow
   2) bubbly
   ...

  s) Статус окон
  a) Подключиться к tmux
  q) Выход
```

### Выбор проекта

```
→ 1                          # Выбрать проект по номеру
→ 1 взять задачу 5           # Выбрать проект с задачей
```

### Выбор пайплайна

```
  Доступные пайплайны:
    1) Default Pipeline [default]
    2) Hotfix [hotfix]

  Выбери пайплайн (Enter = 1):
```

### Выбор этапа

```
  Текущая задача: Задача #6 — Редактирование карточек
  Текущий этап: PM_REVIEW

  Этап (Enter = текущий): DEV    # Или Enter для продолжения
```

### tmux

| Действие | Команда |
|----------|---------|
| Подключиться | `tmux attach -t ai-workflow` |
| Следующее окно | `Ctrl+B, N` |
| Предыдущее окно | `Ctrl+B, P` |
| Переключиться на окно N | `Ctrl+B, <номер>` |
| Отключиться | `Ctrl+B, D` |

### Горячие клавиши внутри меню

| Клавиша | Действие |
|---------|----------|
| `s` | Показать статус окон tmux |
| `a` | Подключиться к tmux |
| `q` | Выход |
