Wireframe: Роль Developer та гранулярний контроль доступу (v2)

Low-fidelity wireframe. Перекреслені червоні елементи -- приховані для вказаної ролі. Сині елементи -- видимі, але read-only. Всі 18 User Stories покриті. Дата: 2026-04-05.

Зміст

  1. Tab Navigation (по ролях) -- US-17
  2. Board (по ролях) -- US-3, US-4, US-10
  3. Board -- модал задачі (read-only vs edit) -- US-3, US-10
  4. Pipeline tab (по ролях) -- US-4, US-6, US-11
  5. Terminal tab -- US-5
  6. Queue tab (по ролях) -- US-7, US-12
  7. Backlog tab -- US-9
  8. Agents tab (по ролях) -- US-13
  9. Members list (по ролях) -- US-14, US-15, US-16
  10. Role dropdown (owner only) -- US-14, US-16
  11. Catalog badge -- US-16
  12. 403 Forbidden -- US-9, US-17
  13. Локалізація ролей -- US-18
  14. Бейджі ролей (кольори) -- US-16

1. Dashboard -- Topbar з вкладками (по ролях) [US-17]

Topbar -- Viewer (6 вкладок)
AI Workflow Backlog Board Archive Pipeline Agents Terminal Transcriber Queue Artifacts Members Logs Join Requests User Viewer
Viewer бачить: Board, Archive, Agents, Artifacts, Members, Logs (6 вкладок). Приховані: Backlog, Pipeline, Terminal, Transcriber, Queue, Join Requests, Settings.
Topbar -- Developer (10 вкладок)
AI Workflow Backlog Board Archive Pipeline Agents Terminal Transcriber Queue Artifacts Members Logs Join Requests User Developer
Developer бачить все що viewer + Pipeline (read-only), Terminal, Transcriber, Queue (read-only). Всього 10. Приховані: Backlog, Join Requests, Settings.
Topbar -- Editor (11 вкладок)
AI Workflow Backlog Board Archive Pipeline Agents Terminal Transcriber Queue Artifacts Members Logs Join Requests User Editor
Editor бачить все що developer + Backlog. Всього 11. Pipeline та Queue -- повний доступ. Приховані: Join Requests, Settings.
Topbar -- Owner (13 вкладок)
AI Workflow Backlog Board Archive Pipeline Agents Terminal Transcriber Queue Artifacts Members Logs Join Requests User Owner
Owner бачить все: 13 вкладок, включаючи Join Requests та Settings. Повний контроль над проектом.
