# Задача #1

**Проект:** comfy-agents
**Контекст проекта:** ../projects/comfy-agents/project.md
**Источник:** backlog #1, приоритет high

## Описание

Баг классификации: пользователь нажимает навигационную карточку "Підбери товар" и пишет "треба гарний роутер". Оркестратор отправляет запрос в FAQ-агент вместо product-agent.

## Ожидаемое поведение

Когда пользователь активировал функционал "Підбери товар" (через навигационную карточку), все последующие сообщения в рамках этого контекста должны маршрутизироваться в product-agent (рекомендации/подбор товара).

## Фактическое поведение

Сообщение "треба гарний роутер" уходит в FAQ-агент.
